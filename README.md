# pgn_editor
A desktop program for viewing and editing PGN files (with a focus on opening preparation)

## Development Status
This project is still in the pre-version-numbers alpha.

## Prerequisites
- Python3.12+ (possibly works with Python3.10+, currently untested)
- `chess` & `pygame` (`pip install chess pygame`)
- the Sourcecode Pro font
